# **App Name**: Vizo Agency Hub

## Core Features:

- Service Listing & Filtering: Display service offerings with category filtering and interactive detail modals.
- Portfolio Showcase: Showcase Vizo's projects with categorized filtering and interactive modals.
- Contact Form Submission: Service request form with client-side validation and Convex submission.
- User Authentication: Allow users to authenticate via email OTP or guest login using Convex Auth.
- Cursor-based Pagination: Implement cursor-based pagination for the portfolio items.
- Database storage: Use Convex database to manage `portfolioItems` and `serviceRequests` tables.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to evoke creativity and professionalism.
- Background color: Very dark purple (#1A002E). Provides a dark scheme backdrop that will complement the vibrant primary color.
- Accent color: Electric indigo (#7C4DFF) to highlight key interactions and call to action.
- Body and headline font: 'Inter', a grotesque-style sans-serif suitable for both headlines or body text.
- Use a set of modern and minimalist icons to represent services and categories.
- Implement a clean, responsive layout that adapts to different screen sizes.
- Subtle animations and transitions to enhance user experience and interaction.
